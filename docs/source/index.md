# OpenCyberCity documentation

Welcome to the OpenCyberCity documentation! This repository contains comprehensive documentation for all systems and sub-systems of the OpenCyberCity Testbed project at Virginia Commonwealth University. The documentation includes system architecture, design specifications, user guides, and more.

If you are new to the OpenCyberCity Testbed project, we recommend starting with the "Getting Started" section, which provides an overview of the project and its components. From there, you can explore the various sections of the documentation to learn more about specific systems and sub-systems.
