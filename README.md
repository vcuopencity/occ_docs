# occ_docs

Documentation for all systems and sub-systems of the OpenCyberCity Testbed project at Virginia Commonwealth University. This repository contains the documentation for the OpenCyberCity Testbed project, including system architecture, design specifications, and user guides.

This repo also hosts the Sphinx documentation for the project, which can be found at `<website_placeholder>`. The Sphinx documentation is organized into different sections based on the various systems and sub-systems of the project. Each section contains detailed information about the design, implementation, and usage of the respective system or sub-system.
